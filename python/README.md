# pyMaCh3

## Introduction

Currently this is a barebones implementation of MaCh3 and is missing a bunch of useful features. The code currently lets you get likelihoods, set/get parameter values and initialise MaCh3 from a YAML.

This is not the final release and is designed as a temporary stopgap!!!!


## Functionality
In order to link libraries properly please run `build/bin/setup.MaCh3DUNE.sh`

All python apps are then installed in `build/python/apps` and can be run with 

`python3 /path/to/app [OPTIONS]`

The basic interface is shown in `demo_app.py`.
